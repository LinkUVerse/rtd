Rtd provides developer kits that act as wrappers for the Rtd API. The Rtd community broadens the code coverage with its own set of developer kits targeting the Rtd blockchain. 

## Rtd SDKs

<Cards>
<Card title="dApp Kit" href="https://sdk.linkulabs.com/dapp-kit">
    A web frontend SDK that interacts with the Rtd API. It is available as an NPM package.
</Card>
<Card title="Rust SDK" href="./rust-sdk">
    SDK configuration and examples of using the Rtd API with Rust, using the `rtd-sdk` crate.
</Card>
<Card title="Rust SDK Auto-generated Docs" href="https://linkulabs.github.io/rtd/rtd_sdk/index.html">
    Auto-generated documentation for the `rtd-sdk` crate in the Rtd repository.
</Card>
<Card title="TypeScript SDK" href="https://sdk.linkulabs.com/typescript">
    TypeScript SDK for integrating Rtd in your TS dApps.
</Card>
<Card title="zkSend SDK" href="https://sdk.linkulabs.com/zksend">
    zkSend SDK to enable you to incorporate Stashed functionality.
</Card>
</Cards>

## Community SDKs

:::info 

While the community projects are expertly developed, their maintenance and community support vary. You might want to research a project's history and support level before committing to using its utilities. 

:::

<Cards>
<Card title="dApp Kit (Vue)" href="https://github.com/RtdFansCN/rtdue">
    Rtd dApp Kit for the Vue framework.
</Card>
<Card title="Dart SDK" href="https://github.com/mofalabs/rtd">
    A cross-platform Rtd SDK for mobile, web, and desktop.
</Card>
<Card title="Go SDK" href="https://github.com/block-vision/rtd-go-sdk">
    SDK for developing for Rtd using Golang.
</Card>
<Card title="Kotlin SDK" href="https://github.com/mcxross/krtd">
    Krtd is a collection of Kotlin Multiplatform JSON-RPC wrapper and crypto utilities for interacting with a Rtd full node.
</Card>
<Card title="Python SDK" href="https://github.com/FrankC01/pyrtd">
    pyrtd is a Python client for developing on the Rtd blockchain. 
</Card>
<Card title="Swift SDK" href="https://github.com/opendive/rtdkit">
    RtdKit is a Swift SDK natively designed for developing on the Rtd blockchain.
</Card>
